この画像を分析して、描かれているアニメ・ゲーム・漫画のキャラクターを特定してください。

## 分析対象
- アニメキャラクター
- ゲームキャラクター  
- 漫画・ライトノベルキャラクター
- Vtuber・バーチャルキャラクター

## 特に注意すべき点
- 同人作品でよく描かれる人気キャラクター
- 髪色、目の色、服装、アクセサリーなど特徴的な要素
- ポーズや表情からの判断
- 背景やアイテムからのヒント

## 回答形式
以下のJSON形式で回答してください：

```json
{
  "character_name": "キャラクター名（フルネーム推奨、不明な場合は空文字）",
  "original_work": "原作作品名（アニメ、ゲーム、漫画等の正式タイトル）",
  "confidence": 85,
  "character_features": "キャラクターの特徴的な要素（髪色、服装、装身具等）",
  "art_style": "画像の画風やスタイル（イラスト、3DCG、写実的等）"
}
```

## 信頼度について
- 90-100: 確実に特定できる
- 70-89: 高い確信があるが、類似キャラクターの可能性もある
- 50-69: 推測だが、特徴が一致している
- 30-49: 不確実だが、可能性がある
- 0-29: 特定困難

キャラクター名や作品名が不明な場合は、空文字を設定し、confidence を低く設定してください。
推測や曖昧な情報よりも、確実な情報を重視してください。
